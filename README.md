Дополнение:
все управляющие последовательности удовлетворяют регулярному выражению
```
%([0-+ ]*)(1[0-9]*)?(ll)?(u|i|d|%)
```
Если тип — %, то флаги, ширина поля и размер данных игнорируются, просто выводится знак %.
Например, последовательность
```
%+++ll%
```
допустима и выводит %, а
```
%ll %
```
недопустима.
